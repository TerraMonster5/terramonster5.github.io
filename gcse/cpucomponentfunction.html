Unit One
CPU Components and their Function

CPU Components

The CPU is a component of a computer but it is also made up of smaller components. The CPU has a number of registers which are: Accumulator, Program Counter, Memory Address Register and Memory Data Register. There are 4 more components which make up a Von Neumann CPU, these are the: Cache, Clock, Control Unit and Arithmetic Logic Unit.

Functions

CPU Clock

The CPU clock determines how many Fetch-Decode-Execute cycles happen every second. Modern clocks are tiny crystals which emit and electrical signal very regularly and reliably. One pulse from a clock is call one hertz and most modern CPUs run around 3Ghz which is 3 billion signals per second. With a 3Ghz clock a computer can perform 3 billion fetch, decode or execute functions a second.

Registers

Registers are super fast, tiny pieces of memory onboard the CPU.

Accumulator - The Accumulator stores numbers that are used in the CPU's processes.

Program Counter - The Program Counter stores the program number.

Memory Address Register - The Memory Address Register stores the memory address where data is to be fetched or stored.

Memory Data Register - The Memory Data Register stores instructions that are held at the memory address which is stored in the Memory Address Register. It also stores data which is to be transferred to RAM.

Other Components

Cache - The Cache stores frequently used instructions and data.

Arithmetic Logic Unit - The Arithmetic Logic Unit performs mathematical calculations.

Control Unit - The Control Unit controls how data moves around the CPU.
