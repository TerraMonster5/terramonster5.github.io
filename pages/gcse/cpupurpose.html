The Purpose of the CPU - The Fetch-Execute Cycle

What is a CPU?

CPU stands for Central Processing Unit and is the 'brain' of a computer. The CPU follows a set of instruction which are given to it from Random Access Memory (RAM).

The Fetch-Execute Cycle

- The CPU fetches the next instruction from RAM.

- The CPU decodes the instruction.

- The CPU executes the command.
