What is a Computer System?

A computer system is an electrical system which can perform Input, Process, Output.

What is Input, Process, Output?

For a computer an input coud be a number or the position of a dial. The process is what the computer does with that information. Finally the output is the result of any calculations and processing.

Where do storage and feedback fit in?

Feedback takes the output of a computer and puts it staright back into the input, storage stores information that comes from processing so it can be used in processing again.
